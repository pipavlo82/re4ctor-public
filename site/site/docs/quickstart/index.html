Quickstart

Minimal examples for calling the API.

1) Get randomness

curl -sS "/api/v1/random?n=32&fmt=hex" \
  -H "X-API-Key: YOUR_API_KEY"

2) Browser usage (CORS)

Use X-API-Key header. Preflight is supported for allowed origins.
