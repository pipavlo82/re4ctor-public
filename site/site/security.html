Security

Last updated: 2025-12-21

Reporting

Please report vulnerabilities to pientropy.2025@gmail.com. Include steps to reproduce and impact assessment.

Scope

  • re4ctor.com and api.re4ctor.com public endpoints
  • Authentication, rate limiting, and API gateway behavior
  • Documentation and published artifacts

Out of scope

  • Third-party services (e.g., payment provider)
  • Denial-of-service testing that impacts availability

Response

We aim to acknowledge reports promptly and coordinate a fix and disclosure timeline.
