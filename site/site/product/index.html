Re4ctoR Product Overview

Re4ctoR is a nuclear-grade entropy engine with signed randomness, post-quantum security, and global delivery via Cloudflare's edge. Designed for Web3, fintech, defense and high-integrity systems.

Core Features

High-entropy core
Multi-layer entropy: Xoshiro, ChaCha20, Keccak, Fisher-Yates, Pi-noise, system jitter, template analysis, bias correction and whitening.
Ultra-low latency
  • <1 ms — inside the machine (local mode)
  • 20–30 ms — global SaaS via Cloudflare Anycast
950k requests/sec (local)
Optimized parallel pipelines allow near-hardware throughput.
Verifiable randomness
  • ECDSA(secp256k1) signatures
  • ML-DSA / Dilithium (post-quantum, roadmap Q2 2026)
  • Hybrid dual signatures
Deployment options
  • SaaS via Cloudflare
  • Self-hosted (Docker / systemd)
  • Secure Enclave edition
  • Air-gapped / Defense edition (NDA)
FIPS Roadmap
  • FIPS 140-3 entropy source preparation
  • FIPS 204 (post-quantum) alignment
  • Deterministic KAT vectors + test harness

Architecture

Cloudflare → Caddy → API Gateway → Entropy Core

Cloudflare handles global routing, TLS and DDoS protection. Caddy terminates edge TLS (origin). Gateway enforces API keys, signatures and rate limiting. Core produces entropy and executes PQ/dual signatures.

Why Re4ctoR

  • Faster than Chainlink VRF (seconds → milliseconds)
  • Self-hosted unlike drand or Chainlink
  • PQ-secure unlike most existing VRF providers
  • Battle-tested statistical pipeline (Dieharder / STS / PractRand)
